🚀 Web Resource Vault

🛠️

Cómo usar la extensión

1. Abre las Herramientas de Desarrollo (F12)

2. Ve a la pestaña Web Resource Vault

3. Haz clic en Guardar en Vault para descargar todos los recursos

💡

Características principales

✅ Descarga completa de sitios web con estructura original

✅ Filtros avanzados por tipo de archivo y tamaño

✅ Vista previa de recursos antes de descargar

✅ Compresión inteligente y caché optimizado

⚠️

Nota importante

La extensión no está disponible en páginas de chrome:// o en modo incógnito por limitaciones de seguridad del navegador.
