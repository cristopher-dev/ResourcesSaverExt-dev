📚 Instrucciones

🛠️

Entorno de Herramientas de Desarrollo

La extensión funciona en el entorno de herramientas de desarrollo para leer la información de recursos. Necesitarás reiniciar tu navegador o reabrir el panel de herramientas de desarrollo, y cambiar a la pestaña Web Resource Vault.

🚫

Limitaciones de URL

La pestaña ResourceSaver no estará disponible en el panel de herramientas de desarrollo mientras tu pestaña actual esté en la URL https://chrome.google.com/*

🕵️

Modo Incógnito

No funciona en modo incógnito. Necesitas reiniciar el panel de herramientas de desarrollo para cambiar de versión.
